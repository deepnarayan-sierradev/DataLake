Technology Stack Reference

LayerTechnology
CloudAWS (EventBridge, Step Functions, Lambda, ECS Fargate, S3, DynamoDB, Secrets Manager, Glue, Athena, RDS MySQL, SQS, CloudWatch, X-Ray, SNS, KMS, IAM, VPC)
Data formatApache Parquet (Snappy) for data layers; JSON for configs and reports
Source APIsSalesforce Bulk API 2.0; NetSuite SuiteQL REST; MySQL JDBC via pymysql
LanguagePython 3.14 — Pydantic v2, structlog ≥ 24.4, pyarrow, boto3, pymysql, requests
InfrastructureTerraform ≥ 1.8 (AWS Provider ~> 5.0); 3 environments: dev / staging / prod
CI/CDGitHub Actions — 7-stage gate: Ruff → mypy → pytest → bandit → pip-audit → checkov → terraform validate
Security algorithmsHMAC-SHA256 (PII tokenisation); SHA-256 (field fingerprinting); Jaro-Winkler + Jaccard (entity resolution)
Testingpytest + moto ≥ 5.0 (AWS mocking); ≥ 80% coverage gate enforced in CI
